### JavaScript

- Array.some

  - 특정 조건을 만족하는 값이 하나라도 있다면 true를 반환함
  - ex)

  ```Javascript
  var arr = [2,3,4];
  var t = arr.some(x => x > 3)
  console.log(t) // true
  ```
